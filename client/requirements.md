## Packages
framer-motion | Smooth animations for feed swiping, modals, and XP celebrations
react-player | Universal media player for YouTube, TikTok, Vimeo, etc.
canvas-confetti | Particle effects for completing a learning item
@types/canvas-confetti | Types for canvas-confetti
react-intersection-observer | To detect when a feed card is snapped into view to mark it as "surfaced"
date-fns | Date formatting

## Notes
Dark mode deep navy theme is set as the default in index.css.
Feed uses CSS scroll snapping (snap-y snap-mandatory) to simulate TikTok feed feel.
